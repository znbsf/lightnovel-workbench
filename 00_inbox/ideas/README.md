# 点子库

这里存放还没有进入正典的原始灵感。

“点子”可以是：

- 一段气氛文字
- 一个画面
- 一种人物关系
- 一个开场
- 一句对白
- 一个“如果这样会怎样”的设想

这里的内容默认都不是正典。

也就是说：

- 可以互相矛盾
- 可以很短
- 可以只有感觉，没有完整剧情
- 可以只保留你现在最想抓住的那股劲

建议规则：

1. 一条点子一个文件。
2. 先保留原始文字，再做整理，不要一上来就改写得太完整。
3. 点子被正式采用后，再把它拆进 `01_bible/`、`02_world/` 或 `03_outline/`。
4. 如果点子已经失效，也不要急着删，可以改成 `archived`，以后还能回看。

推荐状态：

- `inbox`：刚收进来，还没整理
- `incubating`：觉得有戏，正在发酵
- `promoted`：已经部分升格到正典文件
- `archived`：暂时不用，但保留
