# 佐料风格说明

这个目录用于存放“局部风味文件”。

它的作用不是替代全书文风，而是在某个章节、场景或片段里，临时加入更具体的气质，例如：

- 某一场恋爱喜剧对话想更轻、更跳、更有错位感
- 某一场战斗想更冷、更利落、更有镜头切换感
- 某一段独白想更安静、更稀薄、更接近青春伤感作品的空气感

## 使用原则

1. 先有 `01_bible/style.md`，再有局部风味。
2. 一个场景尽量只挂 1 个佐料风格文件，最多不超过 2 个。
3. 佐料风格只调“怎么写”，不改“写什么”。
4. 佐料风格主要影响语气、节奏、镜头感、意象偏好和笑点方式。
5. 不要把它写成“完全模仿某作者或某作品”，而要抽取可复用的特征。

## 推荐命名

- `romcom_light_banter.md`
- `youth_melancholy_soft_focus.md`
- `battle_cold_precision.md`
- `mystery_unease_closed_room.md`

## 使用位置

- 在场景卡的“佐料风格文件”里填写对应文件名
- 在“混合强度”里写明轻度、中度、重度，或直接写比例

## 自检问题

- 这一场是否还是这本书，而不是突然变成另一本书？
- 风味是否只增强了局部效果，而没有改掉人物说话方式？
- 读者能不能感觉到变化，但不会出戏？
